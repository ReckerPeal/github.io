搭建Hexo发布文章的一些问题

常用命令

  1. init，初始化一个新的站点，如果没有设置 folder ，Hexo 默认在目前的文件夹建立网站

  2. new :新建一个文章， layout指对应目录，title指文章标题

    新建一篇文章。如果没有设置 layout 的话，默认使用 _config.yml 中的 default_layout 参数代替。如果标题包含空格的话，请使用引号括起来。

    参数 描述
    -p, --path 自定义新文章的路径
    -r, --replace 如果存在同名文章，将其替换
    -s, --slug 文章的 Slug，作为新文章的文件名和发布后的 URL

  3. 打包 ：hexo g

  4. 启动服务：hexo s
